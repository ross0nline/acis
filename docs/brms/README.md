## The One-Paragraph Summary

ACIS (Autonomous Compliance Intelligence System) is a production-deployed, AI-driven compliance operations platform built on Cloudflare's serverless infrastructure. It monitors five federal regulatory sources daily, tracks client RxDC and Gag Clause attestation status, manages vendor security risk with real-time scanning, auto-generates NIST 800-61 incident response playbooks, and audits its own health every morning — reporting to a central administration layer via zero-latency service bindings. Every AI reasoning trace is logged and visible. The system is live now at the URL above.

This was built to demonstrate what a Security Compliance Administrator II can accomplish when they bring engineering depth to the role. The position description asks for someone who manages compliance programs. This candidate builds the infrastructure that makes those programs autonomous.
